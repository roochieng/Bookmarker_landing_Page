Guide About Features

Features

Here are the Features the YouTube Video Bookmarker currently has:

User account creation and login
Adding bookmark

This is the feature enabling user to save a video

User watching the bookmarked video at anytime

User clicks on the video name to open the video in YouTube

Users are able to delete a video.

When done, user clicks on the delete button to delete the bookmark.

Users cannot duplicate bookmarks.
Users passwords are hashed to ensure security.
